# 快速生成项目目录结构

## 使用npm安装treer模块来生成

安装模块
```bash
npm install -g treer
```

用法：
```bash
在当前目录的终端执行下面命令
# 忽略某些文件或文件夹
treer i "node_modules"

# 保存目录结构到文件
treer -e "自定义文件名.txt"

# 忽略某个文件夹 并输出到制定文件中
treer -i "node_modules" -e "dd.md"
```
